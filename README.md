# Love generator

### Идея проекта

Приложение создано для обмена лайками между пользователями определенной соцсети. 
Идея состоит в том что пользователь авторизируется через определенную соц сеть 
и получает возможность лайкать фотографии других пользователей, 
таким образом повышая свой личный рейтинг. Рейтинг же в свою очередь 
влияет на вероятность и частоту появления фотографий пользователя у других учасников.

### Технические требования: 

Android 5+, Social network account(facebook, instagram, vk)


### Потенциальная аудитория


Активные пользователи соц сетей и те кто желает получить больше популярности.
